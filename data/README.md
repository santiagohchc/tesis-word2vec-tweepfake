
# Datos

Este proyecto usa el dataset [**TweepFake** (Fagni et al., 2021).](https://arxiv.org/abs/2008.00036)

No se redistribuye aquí por respeto a su licencia. Descárgalo desde:
https://github.com/tizfa/tweepfake_deepfake_text_detection/tree/master/data/splits

Coloca los archivos en esta carpeta con la siguiente estructura:
data/train.csv
data/test.csv
data/validation.csv

Referencia:
Fagni, T., Falchi, F., Gambini, M., Martella, A., & Tesconi, M. (2021).
